# FinSight AI - Frontend Dashboard 🖥️

Built with **React 18**, **Vite**, **TailwindCSS**, and **Redux Toolkit**.

## Setup & Running
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local server:
   ```bash
   npm run dev
   ```

The client dashboard runs locally on `http://localhost:5173`.
